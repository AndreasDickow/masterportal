FHH - Flächeninformationssystem - Kurzinformation X

Das FHH-Flächeninformationssystem ist als interaktive Kartenanwendung aufgebaut. Die Kartenanwendung und die integrierte Berichtsfunktion bieten einen umfassenden Überblick über die relevanten Grundstücksinformationen, die für die Entwicklung von Flächen sowie das strategische Flächenmanagement benötigt werden. Die Berichtsfunktion bietet eine Zusammenstellung der Daten zu einzelnen Flurstücken. Über einen Klick in der Kartenanwendung können punktuell Informationen abgerufen werden. Als Datenquelle dienen in der Stadt oder Metropolregion Hamburg verfügbare und im FHH-Intranet freigegebene Daten.

Themen

Einzelne Datenthemen (Layer) können durch Anhaken in der Kartenansicht Dargestellt werden. Über den I-Button werden Informationen aus dem Hamburger Metadatenkatalog (HMDK) und die Legende der einzelnen Layer angezeigt. Weitere Informationen zu einem Layer können über einen Doppelklick auf den dargestellten Layer abgerufen werden.

Grundfunktionen

Folgende Grundfunktionen/Werkzeuge stehen zur Verfügung: Zeichnen/Schreiben, Strecke/Fläche messen, Koordinaten abfragen, Drucken, KML Import, Auswahl speichern, Legende, Kontakt, Informationen (Bodenrichtwertauskunft, Leitungsauskunft) und ein Hilfemenü.

Adresssuche und Flurstücke

Die gezielte Suche nach einer Adresse oder einem Flurstück (Gemarkungsnummer gefolgt von der Flurstücksnummer, mit/ohne Leerzeichen) ist über die Eingabemaske (Suche) oberhalb der Kartendarstellung möglich. Zum Anzeigen von zugehörigen Informationen wird in der Karte ein Flurstück per einfachen Klick ausgewählt (ab Maßstab 1:5000).

Berichtsfunktion

Durch Auswahl eines Flurstücks (ab 1:5000 / Darstellung rot markiert) und Auswahl des Buttons „Bericht erzeugen“ kann der mehrseitige Ergebnisbericht erstellt werden (kann evtl. einige Minuten dauern) und ggf. gespeichert oder ausgedruckt werden. Der Bericht enthält sowohl Felder, die im Rahmen der aktuell verfügbaren Datenbasis gefüllt werden, als auch Felder, die erst in Zukunft bedient werden können. Letztere werden im PDF-Dokument durch den Platzhalter dargestellt „derzeit keine Daten verfügbar“. Hinter „leeren“ Feldern hingegen liegt eine Datenbank, die aber zu diesem Thema (z.B. Schutzgebiete) keinen Eintrag liefert.
