staticconfig.allowParametricURLBoolean

Wenn TRUE, wird in main.js models/ParametricURL.js geladen. Dieses Modul übernimmt spezielle Attribute eines parametrisierten Aufrufs und überschreibt damit Einstellungen der config.js

staticconfig.attributionsBoolean

Wenn TRUE, wird in main.js views/AttributionView.js geladen. Dieses Modul regelt die Darstellung der Layerattributierung aus layerConf oder layerIDs{attribution}.

staticconfig.isMenubarVisibleBoolean

Steuert, ob die Menubar initial ausgeklappt ist oder nicht.

staticconfig.layerConfString

Pfad zur DienstAPI.

staticconfig.layerIDsArray.<Object>

Beschreibung.
Properties:
Name Type Description
id String ID aus layerConf. Werden kommaseparierte ID übergeben, können WMS gemeinsam abgefragt werden.
visible Boolean Initiale Sichtbarkeit des Layers.
style String Nur bei WFS-Layern. Weist dem Layer den Style aus styleConf zu.
styles String Nur bei WMS-Layern. Fragt dem WMS mit eingetragenem Styles-Eintrag ab.
clusterDistance Number Nur bei WFS-Layern. Werte > 0 nutzen Clustering.
searchField String Nur bei WFS-Layern. Wenn searchField auf Attributnamen gesetzt, werden die entsprecheden Values in der Searchbar gesucht.
styleField String Nur bei WFS-Layern. Wenn styleField auf Attributname gesetzt, wird der jeweilge Wert für Style benutzt. styleConf muss angepasst werden.
styleLabelField String Nur bei WFS-Layern. Wenn styleLabelField auf Attributname gesetzt, wird der jeweilge Wert für Label verwendet. Style muss entsprechend konfiguriert sein.
mouseHoverField String Nur bei WFS-Layern. Wenn mouseHoverField auf Attributnamen gesetzt, stellt ein MouseHover-Event den Value als Popup dar.
filterOptions Array.<Object> Nur bei WFS-Layern. Array aus Filterdefinitionen. Jede Filterdefinition ist ein Objekt mit Angaben zum Filter.
Properties
Name Type Description
fieldName String Name des Attributes, auf das gefiltert werden soll.
filterType String Name des zulässigen Filtertyps. Derzeit nur combo.
filterName String Name des Filters in der Oberfläche.
filterString String Einträge des Filters, auf die gefiltert werden kann.
attribution String Setzt die Attributierung des Layers auf diesen String.
attribution Object Setzt die Attributierung des Layers in Abhängigkeit eines Events. Eine Funktion muss den Value 'eventValue' am Layer setzen, um ihn zu übernehmen.
Properties
Name Type Description
eventname String Name des Events, das abgefeuert wird.
timeout String Dauer in Millisekunden für setInterval.
opacity String Wert für die voreingestellte Transparenz für den Layer.

staticconfig.mouseHoverBoolean

Wenn TRUE, wird in main.js views/MouseHoverPopupView.js geladen. Dieses Modul steuert die Darstellung des MouseHovers entsprechend layerIDs{mouseHoverField}

staticconfig.orientationBoolean

Ermöglicht über einen Button auf der Karter den aktuellen Standpunkt bestimmen zu lassen.

staticconfig.poiBoolean

Vorraussetzung für POI(Points of interest) ist, dass orientation auf true gesetzt ist. POI zeigt alle in der Nähe befindlichen Objekte von eingeschalteten WFS Diensten an in den Abständen 500, 1000 und 2000 Metern.
