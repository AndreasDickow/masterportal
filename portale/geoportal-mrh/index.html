Die wichtigsten Neuerungen auf einen Blick

Update 13.09.2016

Zeichnen / Schreiben
Das Werkzeug wurde um drei Funktionen erweitert:
- Zeichnungen können nächträglich bearbeitet werden
- Löschen einzelner Zeichnungen
- Zeichnungen können als KML-Datei heruntergeladen werden
KML Import
Als neues Werkzeug ist der KML Import dazugekommen. Dadurch können z.B. heruntergeladene Zeichnungen wieder importiert werden
Auswahl speichern
Zusätzlich wird die aktuelle Transparenz der Layer gespeichert.
Vollbild
Über den Fullscreen Button ( rechte Seite), wird der Vollbild-Modus aktiviert. Über den gleichen Button kann dieser Modus wieder verlassen werden.

Update 18.04.2016

SimpleMap
Über den Button Auswahl speichern wird nun auch ein Link erstellt, der den SimpleMap aus dem früheren MRH-Portal ersetzt.
Zeichnen / Schreiben
Das Werkzeug wurde erweitert, um die Funktion gezeichnete Elemente einzeln wieder zu entfernen.

Update 07.03.2016

Koordinatensuche
Als weiteres Format stellt die Koordinatensuche (im WGS84 Koordinatensystem) Dezimalgrad bereit.
Zeichnen / Schreiben
Das Werkzeug "Zeichnen" heißt jetzt "Zeichnen und Schreiben" und bietet als neue Funktion "Text schreiben". Damit können Texte mit variabler Schriftgröße, -art und -farbe auf der Karte platziert werden. Texte lassen sich auch drucken.
Mausposition
Die Koordinaten der aktuellen Position der Maus werden in der linken unteren Ecke angezeigt.
Hintergrund
Im Themenbaum (bei den Hintergrundkarten) befindet sich ein neues Symbol . Damit kann der Hintergrund zwischen grau (standard) und weiß gewechselt werden.

Update 14.01.2016

Strecke / Fläche messen
Dieses Werkzeug wurde um die Auswahl der Einheit [m oder km bzw. m² oder km²] erweitert. Zusätzlich kann auch die Anzahl der Dezimalstellen festgelegt werden.
Map-Marker
Der Map-Marker, der eine gefundene Adresse in der Karte anzeigt, kann über einen kleinen "x-Button" (oberhalb des Markers) wieder entfernt werden.
Koordinatensuche
Als neues Werkzeug ist die Koordinatensuche dazugekommen. Die Suche ist in den Koordinatensystemen WGS84 und ETRS89 möglich.

Update 15.12.2015

Flurstückssuche
Unter Werkzeuge gibt es eine zusätzliche Flurstückssuche. Hier kann die Gemarkung aus einer Liste ausgewählt werden. Die "Schnellsuche" über das Suchfeld ist weiterhin möglich.
Zeichenfunktion
Zeichnungen können transparent geschaltet werden.

Update 13.11.2015

Auswahl speichern
Über den Button "Auswahl speichern" im Themenbaum, kann die aktuelle Auswahl der Kartenebenen (inkl. Kartenausschnitt) gespeichert werden.
Informationen zu den Kartenebenen / Metadaten
Hinter dem "i-Button" der Kartenebenen öffnet sich ein Fenster mit der Beschreibung der Daten, dem Link zum Eintrag im Hamburger Metadatenkatalog und der entsprechenden Legende.
Menü Legende
Die Legende öffnet sich nicht mehr über die gesamte Anwendung, sondern wird in einem Fenster angezeigt.
Titel und Datum im Ausdruck
In den Druckeinstellungen kann optional ein Titel für den Kartenausdruck angegeben werden. Das Erstellungsdatum wird auf jedem Ausdruck mit angezeigt.
